# App Icons

Place your app icons here:

- `app_icon.png` - Main app icon (1024x1024 px recommended)
- `app_icon_foreground.png` - Adaptive icon foreground (432x432 px recommended)

You can generate icons using:
- https://www.appicon.co/
- https://www.figma.com/
- Or run: `flutter pub run flutter_launcher_icons:main`

After adding icons, run:
```bash
flutter pub run flutter_launcher_icons:main
```
